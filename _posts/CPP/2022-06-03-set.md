## 🪐set


```cpp
// (Key = Value)
set<int> s;

// 넣고
// 빼고
// 찾고
// 순회

// 넣기
s.insert(10);
s.insert(20);
s.insert(30);
s.insert(40);
s.insert(50);
s.insert(60);
s.insert(100);

// 빼기
s.erase(40);

// 찾기
set<int>::iterator findIt = s.find(50);
if (findIt == s.end())
	cout << "Can't find" << endl;
else
	cout << "Find" << endl;

// 순회하기

for (set<int>::iterator it = s.begin(); it != s.end(); ++it)
	cout << (*it) << endl;
```
