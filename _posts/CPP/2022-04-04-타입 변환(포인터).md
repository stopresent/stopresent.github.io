## 🙇‍♀️타입 변환 (포인터)




### 🪐타입 변환 (포인터)



```cpp
// Stack [ type(4) dbid(4) dummy(4096) ]
Item item;

// Stack [ 주소(4~8) ] -> Heap [ type(4) dbid(4) dummy(4096) ]
Item* item2 = new Item();
```
포인터로 만들어준 것은 주소값만 있기 때문에 소멸자가 자동으로 만들어지지 않는다. `delete`를 해줘야 한다. 해주지 않으면 메모리 누수가 일어난다.
