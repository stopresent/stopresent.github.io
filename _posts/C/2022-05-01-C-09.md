### 🪐입출력 버퍼

ABC 라고 입력 -> 버퍼 buffer -> ABC 출력

입력할 때마다 한 글자 씩 버퍼로 이동 -> 버퍼에 담긴 내용이 통째로 전달

```c
char c;

while ((c = getchar()) != '.')
{
	putchar(c);
}
```
문자를 입력하면 바로 출력하는게 아니라 모았다가 한꺼번에 출력함
