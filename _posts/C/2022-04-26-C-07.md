### 🪐while 반복 루프에서 scanf()의 반환값 사용하기

입력 받은 숫자들을 합해주는 프로그램

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main()
{
	int sum = 0, num = 0;

	printf("Enter an integer (q to quit) : ");

	while (scanf("%d", &num) == 1)
	{
		sum = sum + num;

		printf("Enter next integer (q to quit) : ");
	}

	printf("Sum = %d\n", sum);

	return 0;
}
```
