## 🙇‍♀️ref,out

* 복사(짭퉁), 참조(진퉁)
  * ref,out 은 매게변수와 함수를 사용할때 각각 적어주어야한다
  * 참조의 개념으로 진퉁을 사용한다

### 🪐복사
```cs
static int AddOne2(int number)
{
    return number + 1;
}
```

### 🪐ref 키워드 활용
```cs
static void AddOne(ref int number)
{
    number += 1;
}
```

### 🪐out 키워드 활용
```cs
static void Divide(int a, int b, out int result1, out int result2)
{
    result1 = a / b;
    result2 = a % b;
}
```
